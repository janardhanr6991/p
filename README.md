This project is a simple text editor built using HTML, CSS, and JavaScript. The editor allows users to select fonts from Google Fonts, adjust font weight and italic settings, and save their settings locally.

Features
Font Family Selection: Choose from a list of available fonts.
Font Variant Selection: Pick different font weights and styles.
Italic Toggle: Switch between normal and italic text styles.
Auto-Save: The editor automatically saves text content and settings using local storage.
Reset Button: Clear all settings and start fresh.
